Introduction

What is Python?

Python is a high-level, interpreted programming language known for its readability and ease of use.

Created by Guido van Rossum and first released in 1991, Python has a design philosophy that emphasizes code readability and a syntax that allows programmers to express concepts in fewer lines of code.

Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Key Features and Advantages

  • Readability: Clear and easy-to-read syntax.
  • Versatility: Used in web development, data science, artificial intelligence, scientific computing, and more.
  • Community Support: Large and active community.
  • Extensive Libraries: A vast standard library and many third-party modules.
Getting Started

Installation

Windows:

To install Python on Windows, follow the steps below:

  1. Download the Python installer from the official Python website.
  2. Run the installer and follow the instructions.
  3. Add Python to PATH during installation.

macOS:

To install Python on macOS, follow the steps below:

  1. Use Homebrew: brew install python
  2. Verify the installation with python3 --version

Linux:

To install Python on Linux, follow the steps below:

  1. Use the package manager: sudo apt-get install python3

Setting up the Python Environment

IDEs: PyCharm, VSCode.

Text Editors: Sublime Text, Atom.

Jupyter Notebooks: Ideal for data science projects.

Hello, World!

print("Hello, World!")

To run your first program, save it in a file with a .py extension and execute it using the Python interpreter.

Basic Concepts

Variables and Data Types

  • Variables: Containers for storing data values 
    x = 10
y = "Hello"
  • Data Types:
    • Integers: Whole numbers, e.g., a = 5
    • Floats: Decimal numbers, e.g., b = 5.5
    • Strings: Text, e.g., c = "Python"
    • Booleans: True or False, e.g., d = True

Operators

  • Arithmetic Operators: +, -, *, /, %, **, // 
    result = 10 + 5
  • Comparison Operators: ==, !=, >, <, >=, <= 
    is_equal = (10 == 10)
  • Logical Operators: and, or, not 
    is_true = True and False
Control Structures

Conditional Statements

if condition:
    # code block
elif another_condition:
    # code block
else:
    # code block

Loops

  • For loops: 
    for i in range(5):
    print(i)
  • While loops: 
    while condition:
    # code block
  • Loop Control:
    • break: Exit the loop.
    • continue: Skip the current iteration.
    • pass: Do nothing.
Functions

Defining Functions

def function_name(parameters):
    # code block
    return value

Example:

def greet(name):
    return f"Hello, {name}"

Arguments and Parameters

  • Positional Arguments: Passed by position 
    greet("Alice")
  • Keyword Arguments: Passed by name 
    greet(name="Alice")
  • Default Parameters: 
    def greet(name="Guest"):
    return f"Hello, {name}"

Return Values

def add(a, b):
    return a + b

Lambda Functions

add = lambda x, y: x + y

File Handling
  • Opening Files:

Decorators

Function Decorators:

def my_decorator(func):
def wrapper():
    print("Something is happening before the function is called.")
    func()
    print("Something is happening after the function is called.")
return wrapper

def say_hello():
print("Hello!")
